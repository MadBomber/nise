#

RabbitMQ has been chosen as the AMQP server for ISE.  This directory contains plugins to
the server that support web-based management of the AMQP server.  The functions provided
by this interface are similar to those provided by the rabbitmqctl command line utility.

The sub-directory 2.1 are the plugins for RabbitMQ-Server version 2.1.*

The sub-directory 2.2 are the plugins for RabbitMQ-Server version 2.2.*

Use the script fc14_install.sh to install these plugins into the proper place within the
RabbitMQ installation.  The script assumes that your user account as sudo privileges to
the installation directory.

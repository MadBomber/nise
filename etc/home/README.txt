##  Desc: Describes the contents and purpose of
##        $ISE_ROOT/etc/home
#

This directory contains files and subdirectories that are considered
a standard ISE User configuration.  When a new ISE User account is
established the script $ISE_ROOT/bin/configure_ise_user_account will
copy the contents of this directory into the $HOME directory.

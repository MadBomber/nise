The templates directory holds to directories and files that
are used by the genise.rb program to generate common ISE components.

The following directories serve the indicated purpose:

ise_job .......... Contains a template file for generating an IseJob using IseJCL
ise_message ...... Contains both Ruby and C++ templates for defining IseMessages
ise_model ........ Contains C++ templates for defining a C++ implementation of an IseModel
ise_project_svn .. Contains the basic directories and files for a standard subversion controlled ISE project
ise_ruby_model ... Contains Ruby templates for defining a Ruby implementation of an IseRubyModel
ise_web_app ...... Contains the directories and files necessary for defining typical web applications
